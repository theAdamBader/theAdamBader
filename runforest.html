RUN FOREST RUN

SOLO/PROCESSING PROJECT

Java

OVERVIEW

What is the game?

Run Forest Run is a scrolling game in which you play as Forest Grump, a tree that is running from a forest fire. The objective is see how far you can get before the fire spirites get you. Can you help Forest run as fast as you can or will the fire get him and itis over.

Why create this game?

The reason to why I took a different direction is because unlike other scrolling games where you see how fast you can get, I also used the movie reference Forrest Gump to create a sweet and playful game; however there is a subliminal message to it too that no matter how far you run, the fire will catch up and burn the forest. No matter how far that poor tree runs, the fire would eventually get poor Forest Grump.

DESIGN

Character Design

The characters were design using Adobe Illustrator as it Illustrator deals with vectors and not pixels. The main character, was drawn as a sprite sheet with different facial reactions and movement; as for the enemies they had a simple design as it was simple to resize and change their colour value; although the design was simple, giving certain feature allowed the players to know which enemy was dangerous and which were hard to avoid.

Level Design

The level design itself, as seen in the picture, shows that when the art ends as it matches the beginning of the artwork ensuring that when it loops it does not break the immersion and also added a few branches to make it seem like the player is running within the forest itself.

Audio

The level was a simple a design which looped each time making it look like it is a stretch canvas and for the audio I used freesound.org to download the tracks and use it on the game

IMPLEMENTATION

Gameplay

You are able to control one character which is the main character, as the main character you are able to walk, jump, and avoid enemies heading towards you and the forest fire behind you, there are four types of enemies, small fire ball, and large fire ball which comes at a slow pace, a blue fire from a above, incase you jump high and a small green fire that heads towards you very fast. The score is tallied by how far you can reach the point and keeps a highscore to encourage others to best the last person's highscore.

Code

The code implemented for this project is a background loop, so ensuring the background loops well without breaking the immersion. The other code is other enemies and their sprites, however in collision the enemies are summoned depending for far you reach and the animation for Forest is randomised as I wanted it to have a varies reactions as the player moves and jump.
