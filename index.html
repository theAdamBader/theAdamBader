University Projects

1st Year Projects

Introduction to Digital Media: Project
The first project was creating a website using WordPress in which I took the leadership role and decided on what roles everyone would take and how we could able to create a site that would not be too complicated as we were just novices. I made sure that the team produced the work needed, when one member failed to fulfill their role due to lack of communication I took their role to ensure to have a site running before the deadline.
Introduction to Game Development: Project
Our project for Game Development was creating a game with a space theme using Unity. I came up with the title and story in which the alien was alienated by its race and decided to hurt them. It is a dark story with no happy endings. In terms of coding, I was assigned to animate and move both the player and enemy as well as AI dectection, the other team member was in charge of the platforms, and level design as for our project manager he was in charge of collision detection and the health system as well as ensuring that everyone had done the work tailored to his needs.
Introduction to Programming: Project
In Introduction to Programming we had to use Proccessing in order to do our final project of the year, using Java language. In my "final" project I created a simple platform game called Run Forest Run where your main objective, as Forest Grump, is to run as fast as you can from a forest fire and see how far you can go! This was an indivual task so that allowed me to explore my creative side and create a game that would be simple to play.
